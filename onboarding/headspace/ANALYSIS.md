## Design System

### Couleurs
| Usage | Couleur |
|-------|---------|
| Primary | Orange #FF8C00 |
| CTA | Bleu #2B7DE9 |
| Background | Gradient jaune-orange (haut) → blanc (bas) |
| Text | Noir / Gris foncé |

### Mascotte
- Cercle orange avec visage minimaliste
- Yeux fermés (lignes courbes), sourire doux
- Anime avec la respiration

### Typographie
- Titres : Sans-serif bold, 28-32px
- Body : Sans-serif regular, 16-18px, gris
- CTA : Semibold, blanc sur bleu

### Composants
- **Pills de sélection** : Fond blanc, checkbox à droite
- **Boutons** : Bleu, coins arrondis ~25px
- **Pagination** : 3 dots, actif = noir

## Points Clés à Retenir

### ✅ À copier absolument

1. **Exercice de calme AVANT toute demande**
   - Respiration / affirmation animée
   - 3-4 secondes suffisent

2. **3 slides de valeur**
   - Features → Social Proof → Science
   - Format digestible

3. **UNE question de personnalisation**
   - Pas 10 questions
   - Réponse immédiate "personnalisée"

4. **Timeline du trial transparente**
   - Élimine l'anxiété du "je vais oublier"
   - Rappel avant facturation

5. **Prix annuel présenté en mensuel**
   - "$69.99 ($5.83/month)"
   - Semble moins cher

### ❌ À ne pas copier

1. **Palette orange/jaune**
   - Trop énergique pour anxiété
   - Préférer bleu/violet (Moonly)

2. **Signup avant value prop**
   - Headspace demande le compte tôt
   - Mieux : montrer la valeur d'abord

---

## Conclusion

Headspace est la **référence gold standard** pour les apps bien-être. Leur onboarding est :
- Émotionnel d'abord (respiration)
- Éducatif ensuite (value props)
- Personnalisé (1 question)
- Transparent (timeline paywall)

Pour Serein, on adapte :
- Palette → Bleu/violet (plus calme)
- Mascotte → Illustration ou animation douce
- Trial → 3-7 jours (plus court mais suffisant)
