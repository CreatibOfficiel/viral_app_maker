## Design System

### Couleurs
| Usage | Couleur |
|-------|---------|
| Background | Bleu nuit très foncé #0D0D1A |
| Primary | Violet #7C3AED |
| Accent | Rose #FF69B4 / Magenta |
| Cards | Gris foncé avec bordure subtile |
| Text | Blanc / Rose clair |
| Premium | Violet/Magenta gradient |

### Typographie
- Titres : Serif élégant, grande taille
- "yourself" en couleur accent (rose)
- Body : Sans-serif, gris clair

### Illustrations
- Style : Semi-réaliste, digital art
- Personnages : Féminins, mystiques
- Éléments : Lune, étoiles, signes du zodiaque

### Composants
- **Toggle gender** : Pill selector (Female/Male)
- **Input fields** : Fond gris foncé, placeholder rose
- **Progress dots** : 7 points en bas
- **Cards pricing** : Fond sombre, bordure subtile, sélection = fond rose

## Points Clés à Retenir

### ✅ À copier pour Serein

1. **Palette de couleurs**
   - Bleu nuit + violet + rose = parfait pour anxiété/calme
   - Feeling premium et apaisant

2. **Illustration d'intro**
   - Premier écran = impression mémorable
   - Personnage qui incarne la sérénité

3. **Privacy reassurance**
   - Box "Your data is private" rassure
   - Particulièrement important pour app bien-être

4. **Loading avec pourcentage**
   - "100%" dans un cercle mystique
   - Crée anticipation avant le paywall

5. **Paywall sans timeline**
   - Plus simple que Headspace
   - Weekly mis en avant (conversion impulsive)

6. **Tab bar avec "Affirmations"**
   - Preuve que c'est une feature clé
   - Notre app = focus 100% sur ça

7. **Features quantifiées**
   - "200+ Affirmations" = valeur perçue
   - "100+ Meditations" = contenu abondant

### ⚠️ À adapter

1. **Weekly par défaut**
   - Moonly pousse le weekly (plus cher à long terme)
   - Nous : pousser l'annuel mais garder weekly comme option

2. **Pas de trial visible**
   - Moonly semble aller direct au paiement
   - Nous : garder le free trial 3 jours

3. **Birth Chart comme bonus**
   - Leur USP pour l'annuel/lifetime
   - Nous : trouver un équivalent (widget? stats?)
